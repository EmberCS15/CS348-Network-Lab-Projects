To run the Project - 
1. Install Python
2. Install pip
3. Install Django
4. Navigate to the folder containing the manage.py file in Command Prompt/Windows PowerShell/Terminal
5. Use command 
	python manage.py runserver ip_of_server_host
6. Click on create new Game. A token is given in the form of a number On the top of the board
7. Use the token to Join the game
